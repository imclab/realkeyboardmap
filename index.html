Just start typing. Press Shift, Ctrl+Alt, and Shift+Ctrl+Alt for additional characters. Copy & paste into other web forms and most word processors.

Comments and questions? For now you can e-mail me...

Based on ideas from Dan Sieradski, Ze'ev Clementson, and Aharon Varady and Efraim Feinstein from the OpenSiddur project. תודה רבה (much thanks) to jQuery.

Requirements

  1. jQuery plugin
  2. Should work with either <textarea> or <input type="text">.
  3. Work with multiple textareas/input on the page.
  4. Home keys are visually distinct.
  5. As user shifts modes, key caps change.
  6. Affordance of how to invoke other modes?
  7. Resizable?
  8. Able to click keys with a mouse. Control keys, too?
  9. Able to click keys by tabbing to them, if you really want (standard HTML buttons).
  10. "Sticky" option?
  11. Separate profile file, support different languages.
  12. Drop-down to select between US English and other mappings.
  13. User should be able to turn it off.
  14. Ctrl-c, Ctrl-v, etc. should work (i.e., standard shortcut keys not interfered with).

Limitations

Native keyboard is US English only (untested on others).
